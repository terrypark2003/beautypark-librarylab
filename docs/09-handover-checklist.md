## 누적 마찰 요인 (개선 과제)
1. **이벤트 산출물 전달 지연** — 가장 반복된 불만(6월 이벤트, 줄기세포 이벤트 등). → 내부 전환의 최대 개선 포인트: 사전 제작·앞당김.
2. **잦은 디자인 오류/재작업** — 시술명·장비·성분·가격 오기, 맞춤법 오타. → 검수 체크리스트 강제.
3. **잦은 담당자 교체** — 6개월~1년 새 다수 교체로 인계 혼선. → 내부 운영은 연속성 확보.
4. **계정 접근 불안정** — 인스타 2차 인증 코드 의존. → 안정적 권한 체계 구축.
5. **외부 환경 리스크** — 경쟁사 의료광고 신고, 블로그 지수 변동, 네이버 지도 개편 등.
6. **예산/한도 관리** — 메타광고 한도도달로 광고 중단 반복. → 모니터링 체계.
