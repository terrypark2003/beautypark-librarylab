# 04. 디자인 규격 · 파일명 규칙 · 공통 푸터

## 이미지 / 배너 규격 (실측 기준)

| 용도 | 규격 | 비고 |
|---|---|---|
| 홈페이지 메인 팝업 | **2400 × 1350** (16:9) | 리오프닝 이후 표준 |
| 홈페이지 가로 배너(기획전 상세) | **1600 × 500** (범위 360~500) | 기획전 상단 배너형 |
| 이벤트 썸네일 ① | **1080 × 540** | |
| 이벤트 썸네일 ② | **가로 960** | |
| 작업 기준 해상도 | **FHD(1920×1080) 기준** | |
| 인스타 게시물 | 정사각 1:1 또는 4:5 (카드뉴스) | 정확 픽셀 ⚠️ 확인필요 |
| 원내 디스플레이 영상 | `.3gp` / `.mp4` | Premiere(.prproj) 기반 출력 |
| 원내 인쇄물 | A4 (데스크 안내판 비치) | |

> 5월 리오프닝 시 홈페이지 이미지가 **4가지 형태**로 운영됨(팝업/배너/썸네일 2종). 배너는 잘림·깨짐이 잦아 여러 차례 재작업 발생 → 등록 후 **실제 화면에서 잘림 확인 필수**.

## 디자인 스타일

- 밝은 톤(병원 이미지에 맞춤). 과한 채도(짙은 빨강 등) 지양.
- 월별 매거진 테마 + 영문 슬로건/폰트 + 이모지 데코.
- 카드뉴스: 1p 표지(+로고) → 이후 좌측 텍스트 정렬.

## 파일명 규칙 (실제 사용 패턴)

| 산출물 | 패턴 | 예시 |
|---|---|---|
| 이벤트 기획안 | `{YYMM} 이벤트 기획안 (수정N).xlsx` | `2502 베러미 이벤트 기획안-수정2.xlsx` |
| 월간 리포트 | `라랩_{브랜드} 리포트_{YY}.{M}월 리포트.pdf` | `라랩_베러미 리포트_26.4월 리포트.pdf` |
| 디스플레이 영상 | `{브랜드}{지점}_{M}월이벤트_디스플레이.3gp` | `베러미범어점_9월이벤트_디스플레이.3gp` / `06뷰티파크 디스플레이용.3gp` |
| 견적/제안서 | `[라이브러리랩] ...제안서.pdf`, `[LL] ...견적서.pdf` | `[LL] 베러미의원 범어점 견적서_24.08.pdf` |

> 브랜드 변경 후에는 접두/명칭을 **뷰티파크** 기준으로 통일 (예: `뷰티파크범어점_{M}월이벤트_디스플레이.3gp`).
