## 이벤트 구성 공식
- **단일 원데이 특가** (1회 체험가, 진입장벽 낮춤)
- **회차 패키지** (3회/5회/10회)
- **요일 한정** (수/목/금 특가 — "예뻐지는 화수목")
- **시그니처 프리미엄 패키지** (리프팅+스킨부스터+스킨케어 묶음)
- **신장비/신제품 런칭 특가**
- **시즌 훅** (명절·발렌타인·빼빼로·수능·크리스마스·주년)

## 주요 시술/장비 사전 (콘텐츠·이벤트 소재 풀)

**리프팅(고강도)**: 울쎄라 / 울쎄라피 프라임(멀츠, 8mm) / 슈링크 유니버스 / 써마지 FLX / 소프웨이브 / 세르프(XERF) / 쿨페이즈 / 올타이트(DLTD) / 토르RF / 버츄RF / 쿨소닉 / 리프테라2 / 인모드(FX/FORMA) / 올리지오 / 힐로웨이브

**스킨부스터/주사**: 리쥬란 HB/힐러 / 엘라비에 리투오(Re2O) / 쥬베룩 / 위코우노(WiQo Uno) / 쥬브아셀(hADM) / 트리필프로(TRIFILL PRO) / 올리핏주 / 골라 / 물광주사 / 백옥주사 / 태반주사 / 뷰티S주사 / 연어주사 / IBPS 부스터필

**필러/보톡스**: 벨로테로(리바이브/인텐스/소프트/볼륨) / 레스틸렌 / 리쥬비엘 / 나보타·제오민·나보타(국산/수입) / 윤곽주사 / 스킨보톡스 / 더모톡신

**토닝/색소/모공**: GD토닝 / 피코토닝 / 헐리우드 듀얼토닝 / 엑셀V 제네시스 / 라라빔·라라필·라라젯 / 수소아쿠아필 / 노세범(아쿠아필+네오빔) / 프락셀 / 크라이오 / 플래티넘 플라필

**기타**: LDM 물방울리프팅 / 제모(여성/남성 풀페이스) / 다한증 보톡스 / 고압산소(IBEX) / 줄기세포(뷰티스템/노바스템)

## 현재 푸시 중인 시술 (2026 Q2 스냅샷, 블로그/대화 확인)
리브랜딩 이후 집중 홍보 중인 시술 — 다음 이벤트 기획 시 우선 후보:
- **소프웨이브**(리프팅), **올타이트**(DLTD 극초단파), **쿨소닉**(HIFU)
- **엘라비에 리투오(Re2O)**, **힐로웨이브**(Dual-HA), **쥬브아셀**(hADM ECM 부스터)
- **줄기세포 주사**(뷰티스템/노바스템)
- 여름 시즌 훅: 자외선·기미·미백·피부 탄력(6월), 윤곽·제모(여름)
