# 02. 업무 범위 & 정기 업무 캘린더

## 업무 범위 (6축 상세)

### 1) 인스타그램 운영
- 주 2~3회 게시(피드/카드뉴스/스토리/하이라이트).
- 콘텐츠 유형: 이벤트/시술 소개/원내소식/정보성 카드뉴스/매거진/후기(REAL REVIEW)/원장 소개/Q&A.
- 게시물당 산출물 = 이미지 세트(보통 4~7장, 신규·이벤트는 최대 11장) + 본문 카피 + 해시태그 세트.
- 매거진형은 월별 테마(예: "October magazine", "November Glow Up", "Skin Gourmet").

### 2) 카카오톡 채널
- **채널 소식**(피드 게시): 월간 이벤트, 신장비 등.
- **플친 CRM 메시지**: 친구추가 고객 타겟 발송. 와이드 리스트형(항목 최대 3개)/기본형/캐러셀 피드형.
- 자세한 절차는 [06-kakao-crm-playbook.md](06-kakao-crm-playbook.md).

### 3) 월간 이벤트 + 디자인 + 웹 게시
- 핵심 파이프라인. [03-monthly-event-pipeline.md](03-monthly-event-pipeline.md) 참조.

### 4) 메타 유료광고
- 월간 이벤트 홈페이지 링크 받아 광고 세팅. 심의완료 이미지를 광고 소재로 활용.
- A/B 테스트(A=이벤트 문구 강조 / B=브랜드 강조). 단일 이미지가 캐러셀보다 전환율 높다고 판단해 단일 이미지 권장.
- 타게팅: 범어역 근방 3~4km, 핵심 20대 후반~30대 중반. CPM 상승 시 권역 확대.
- 의료광고 신고 리스크로 **최종 퍼널을 카카오톡 상담으로 유도**하는 전략 사용.

### 5) 월간 리포트 + 콘텐츠 일정표
- 매월 중순 전월 성과 PDF 리포트(`라랩_..._리포트`). 친구수 증가, 인스타 도달률, CTR, 카톡 유입 등 지표 코멘트.
- 매월 익월 콘텐츠/디자인 캘린더(이미지) 전달.

### 6) 원내소식 / 안내물
- 신장비 도입, 세미나·심포지엄 참석 소식(병원이 사진/내용 제공 → 카피 작성 후 발행).
- 명절·공휴일 휴진/단축진료 안내물(인스타 + 홈페이지 팝업 + 원내 데스크 A4 출력 2종).
- 네이버 스마트플레이스 썸네일(GIF) 관리, 플레이스 노출/크롤링 모니터링.

### (추가 확인) 네이버 블로그 — 2개 운영 중
- **공식 블로그** `blog.naver.com/betterme_beomeo` (대표원장 칼럼, 400+ 게시물).
- **SEO 마케팅 블로그** `blog.naver.com/drinjectable` ("대구+시술명" 키워드 정보성 글, 주 1~2회).
- 과거 단톡방 로그에선 라이브러리랩 업무 범위에서 블로그가 명확히 확인되지 않았으나, 실제로는 활성 채널 2개가 존재.
- ⚠️ 운영 주체(원내/대행/외주) 확인 후 인수 대상 포함 여부 결정. 포함 시 주 1~2회 키워드 포스팅이 추가 업무가 됨.

## 정기 업무 캘린더

### 월간 (이벤트 사이클) — 라이브러리랩 공식 5단계
| 단계 | 담당 | 마감 |
|---|---|---|
| 1. 익월 이벤트 아이디어 기획 | 라랩(→에이전트) | 매월 첫째 주 화요일 |
| 2. 익월 이벤트 기획안 작성(원내 검수) | 실장 | 매월 12일 |
| 3. 익월 이벤트 POP/배너/팝업 디자인 | 라랩(→에이전트) | 매월 말일 |
| 4. 작업물 확인 + 영상/리사이징 | 라랩/실장 | 매월 말일 |
| 5. 원내 게시 + 웹/SNS 런칭 | 실장 + 라랩 | 익월 1일(평일 기준) |

> ⚠️ **이벤트 산출물 전달 지연이 반복적 마찰 요인**이었음(특히 6월 이벤트, 줄기세포 이벤트). 내부 전환 시 익월분 제작을 **더 앞당기는 것**이 핵심 개선점.

### 주간
- **인스타그램**: 주 2~3회(월·수·금 중심). 전일 검수요청 → 익일 오전 10시경 발행.
- **카카오 플친 메시지**: 주간~격주. 전일(또는 금요일) 테스트 → 익일 오전 11시 예약발송.
  - 2026.5월~ **수요일 테마 메시지** 운영(예: 'Golden-Time 피부관리', '피부 컨디션 리셋', '여름 윤곽케어', '미백관리').

### 월간 (정기 보고)
- 전월 리포트 PDF: 매월 중순(예: 1월분 2/23, 4월분 5/28).
- 익월 콘텐츠 일정표: 매월(리포트와 함께 또는 별도).

### 비정기
- 원내소식(소재 제공 시), 신장비 런칭 POP/팝업, 명절·공휴일 안내물.
