# 07. 검수/승인 절차 & 의료광고 규제 대응

## 검수/승인 절차

### 공통 원칙 (2024-12-11 오타 사고 이후 의무화)
> **모든 게시물은 발행 전 단톡방에 이미지+텍스트를 사전 공유하고 컨펌을 받은 뒤 발행한다.**

### 승인 주체 분담
| 영역 | 검수자 |
|---|---|
| 최종 승인 / 의학적 판단 | 박주현 대표 원장 |
| 시술 내용·장비·성분 디테일 | 박석환 원장 |
| 이벤트 가격·운영·노출 | 최현진 실장 |

### 수정 정책
- 작업물 수정은 **2회까지** (오타·정보 수정 제외).
- 원내 의견은 **한 번에 취합**해 전달 (산발적 요청 지양).
- 완료 후 수정은 영상·베리에이션물 전체 재작업을 유발 → **확정 전에 모든 수정사항 반영**.

### 채널별 검수 포인트
- **인스타**: 시술명/성분/출력 정확성, 맞춤법, 의료법 표현.
- **플친**: 테스트 발송 후 링크·랜딩 연결 확인.
- **이벤트물**: 1차안 → 가격/문구/디자인 피드백 → 최종 확정 → 베리에이션.

## 의료광고 규제 대응 (상시 변수)

- **경쟁사 신고**가 반복됨 → 광고·게시물 일부 임시 삭제, 정보성 텍스트로 전환한 이력.
- **의료심의**: 메타광고는 심의 후 진행(심의 소요 ~1개월). 심의필 이미지를 병원이 제공(zip/사진) → 대행사가 광고 소재로 교차 사용.
- **가격/할인 문구 노출 제한**: 인스타 등 공개 채널 가격 노출 자제. 클릭 유도용 팝업도 가격 미표기 버전 유지.
- **퍼널 전략**: 공개 채널 → **카카오톡 상담**으로 최종 유도(신고 리스크 회피).
- **표현 순화**: "세미나/학회"→"교육 세션", "전문 사용자 모임" 등. 시술 기전·장비 정확 표기(예: 올타이트는 HIFU+RF가 아니라 DLTD; 보유하지 않은 장비/제품 표기 금지).
- **유명인/IP 사용 주의**: 드라마 포스터·배우 이미지 직접 사용 금지. 텍스트로만 표기(예: "OOO 모델 장비").
