- 계정: **`beautypark_beomeo`** — https://www.instagram.com/beautypark_beomeo/
- 특이사항: **잦은 로그아웃 / 2차 인증** 이슈가 반복됨. 로그인 유지를 위해 원내에서 문자 보안코드·백업코드를 그때그때 전달하는 방식으로 운영. 과거(베러미 시절) 해킹·결제 사고 이력 있어 **보안 민감**.
- 운영 시 2단계 인증 백업코드를 안전하게 보관할 것.

### 카카오톡 채널 (플러스친구)
- 채널: **`@뷰티파크의원`**
- (구 채널 URL 패턴: `pf.kakao.com/_UUxdVG/{게시물ID}` — 채널 승계 여부 ⚠️ 확인필요)
- 권한은 **관리자 초대** 방식으로 부여(아이디 공유 아님).

### 메타 비즈니스 (인스타/페이스북 광고)
- 운영 방식: 병원 계정에 대행사를 **파트너 권한**으로 등록(아이디 공유 아님). 인증 강화로 "관리자 일일 초대 → 세팅 → 이탈" 패턴도 사용.
- 광고 재집행 시 새 파트너 계정 등록 필요.
- 월 예산 이력: 초기 월 50만/일 2만 → 이후 약 $300/월 수준. **결제·지출한도 관리는 병원 명의** (한도도달 시 광고 중단되므로 모니터링 필요).
- 페이스북 페이지 운영. 픽셀 설치됨.

### 웹사이트 / 기획전 빌더
- 메인: **https://beautyparkclinic.co.kr/**
- 이벤트 페이지 URL 구조: `/event-introduction?eventIntroductionDurationId={N}&eventIntroductionCategoryId={N}&keyword=`
  - `DurationId` = 기간(월), `CategoryId` = 이벤트 카테고리. 노출/비노출 토글 가능.
- 기획전(쇼핑몰형) 빌더: **Vercel 호스팅 `boostree`/`catalog` 솔루션** 추정. 예: `https://boostree-seed-6-catalog6-2.vercel.app/exhibitions/{id}`
- **배너/팝업 등록**: 대행사가 사이즈별 이미지 제작 후 **홈페이지 관리자에 직접 접속해 등록**하고, 잘림/그리드를 실시간 확인하며 조정했음. → 즉 디자인 + CMS 등록까지 대행 범위.
