# 00. 전체 개요

## 라이브러리랩이 대행하던 업무 = 6개 축

1. **인스타그램 운영** — 주 2~3회 게시물(피드/카드뉴스/스토리/하이라이트). 이미지+카피+해시태그 제작 → 원내 검수 → 발행 → URL 회신.
2. **카카오톡 채널** — ① 채널 소식(피드) 게시 ② 플러스친구 CRM 메시지 발송(친구추가 고객 타겟). 테스트 발송 → 검수 → 오전 11시 예약 발송.
3. **월간 이벤트 기획 + 디자인 + 웹 게시** *(핵심 파이프라인)* — 월간 프로모션 기획 → 원내 POP, 홈페이지 팝업/배너/썸네일, 디스플레이 영상 제작 → 웹/SNS 게시.
4. **메타(Meta) 유료광고** — 인스타/페이스북 광고 집행. 월간 이벤트 페이지로 트래픽 유도.
5. **월간 리포트 + 콘텐츠 일정표** — 전월 성과 PDF 리포트 + 익월 콘텐츠 캘린더.
6. **원내소식 / 안내물** — 신장비 도입·세미나 소식, 명절·공휴일 휴진/단축진료 안내물.

## 운영 골격 (한 장 요약)

```
[월 사이클]
 전월 하순 ── 익월 이벤트 기획 → POP/영상/웹배너 1차안
            → 원내 검수(가격·시술명·문구) → 수정 → 최종 확정
 매월 1일 ── 인스타 + 카카오 + 메타광고 동시 런칭, 홈페이지 배너/팝업 교체
 매월 중순 ── 전월 마케팅 리포트(PDF) + 익월 콘텐츠 일정표 전달

[주 사이클]
 인스타  ── 주 2~3회 (월/수/금 중심). 전일 검수요청 → 익일 오전 발행 → URL 회신
 플친    ── 주간~격주. 전일 테스트발송 → 검수 → 익일 오전 11시 예약발송
            (2026.5월~ 수요일 테마 메시지 운영)
```

## 모든 발행물의 공통 루프 (Closed Loop)

> **제작 → 단톡방에 이미지+텍스트 사전 공유 → 원장/실장 검수·수정 → 최종 확정 → 발행 → 발행 URL을 단톡방에 회신**

이 루프는 2024-12-11 오타 사고(인스타 게시물에 "배러미", "추우수", "찾오시는" 등 오타 연발) 이후 **모든 게시물 사전 컨펌이 의무화**되면서 정착됐습니다. 빠짐없이 지켜야 하는 원칙입니다.

## 담당 역할 분담 (라이브러리랩 기준)

| 역할 | 담당(라랩) | 대체 후 |
|---|---|---|
| 계정관리/PM, 카톡·메타·리포트·일정표 | 고현준 | 에이전트 + 운영자 |
| 인스타 콘텐츠 카피·발행·검수창구 | 배진영 | 에이전트 + 운영자 |
| 디자인(이미지/POP/팝업/배너/영상) | 함지현 → 주은 | 에이전트(생성) + 검수자 |

> 라이브러리랩은 6개월~1년 사이 담당자 교체가 잦았음(신윤서→고현준→남인식→장연호→함지현→배진영→이나연→정세희→주은). 내부 전환의 장점은 **담당자 연속성과 즉시 대응**입니다.
