# 06. 카카오톡 채널 + 플친 CRM 플레이북

채널: **`@뷰티파크의원`**

## 두 가지 채널 활동

### 1) 채널 소식 (피드 게시)
- 월간 이벤트, 신장비 도입 등을 채널 피드에 게시.
- 인스타 콘텐츠와 연동(같은 소재 재활용 가능).

### 2) 플러스친구 메시지 (CRM)
- 친구추가 고객 타겟 발송. 내원 유도/이벤트 알림.
- 포맷: **와이드 리스트형**(항목 최대 3개) / 기본형 / **캐러셀 피드형**.
- 2026.5월~ : 캐러셀에 들어갈 이벤트 항목을 **병원이 3~5개 추려주는 방식** + **수요일 테마 메시지** 운영.

## 발송 표준 절차 (고정)
```
발송 전일(또는 금요일)
   → 테스트 메시지 실발송 (원장/실장 핸드폰)
   → 원장 확인 "이상 없으면 진행"
   → 익일 오전 11시 예약 발송   (오후 1시/3시 변형 가능)
```
> 직장인 점심시간(12~13시) 타겟 시 오전 11시 발송이 표준.

## 메시지 구성
- 헤드라인(테마) + 이벤트 항목 카드(시술명/혜택) + 링크 버튼.
- **버튼 2종**: 홈페이지 버튼 + 이벤트 확인 버튼. 발송 전 **링크 클릭 → 랜딩 연결까지 검수**.
- 가격은 정상가→할인가(→부가세 포함가) 3단 표기 패턴.

## 발송 데이터 추출 예시 (실제)
```
[노세범케어] 블랙필+네오빔+모델링팩 1회 120,000 → 100,000
[화수목] IBPS 부스터필 3회 237,000 → 180,000
이마필러 벨로테로 볼륨 3cc 800,000 → 700,000 (→770,000 VAT)
프리미엄 비타민 미백관리 1회 60,000 → 40,000 (→44,000 VAT)
```
