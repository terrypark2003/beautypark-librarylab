# 05. 인스타그램 운영 플레이북

계정: **`beautypark_beomeo`** (https://www.instagram.com/beautypark_beomeo/)

## 발행 주기
- 주 2~3회 (월·수·금 중심). 금요일은 업로드 고정 슬롯 존재.

## 콘텐츠 유형 (로테이션)
| 유형 | 내용 |
|---|---|
| 이벤트 | 당월 프로모션. 카드뉴스 형태(최대 11장) |
| 시술 소개 | 보톡스/필러/리프팅/스킨부스터 등 (3분할 시리즈 운영) |
| 신장비·신제품 런칭 | 정품 인증, 원리 설명 |
| 원내소식 | 세미나·심포지엄 참석, 신장비 도입, 위생/교육 |
| 정보성 카드뉴스 | FACT CHECK, "무엇이든 물어보세요", Q&A |
| 후기 | "REAL REVIEW" 월별 모음 |
| 매거진 | 월 테마형(예: "Glow Up", "Skin Gourmet") |
| 원장 소개 | 약력·전문성 |

## 게시물 1건 표준 제작
1. **이미지 세트** — 1p 표지(브랜드/테마) → 본문 카드 → 마지막 푸터 카드. 보통 4~7장.
2. **본문 카피** — 후킹 헤드라인 → 효과 불릿 → "이런 분께 추천" → 푸터([04](04-design-specs.md)).
3. **해시태그 세트** — 고정 + 시술별 가변.

## 발행 루프 (필수)
```
전일(또는 당일 오전) 이미지 + 텍스트 단톡방 공유
   → 원장/실장 검수 (시술명·출력·성분·맞춤법)
   → 수정 반영
   → 발행 (익일 오전 10시경)
   → 발행 URL 단톡방 회신
```

## 검수 빈출 포인트 (의학적 정확성)
- 시술 명칭/장비명 정확성 (보유 장비만 표기 — 미보유 장비/제품 표기 금지).
- 출력·성분 표기 ("고출력→저출력", 성분 정식명 병기 등).
- 의료법 표현 순화: "세미나/학회"→"교육 세션", "마취 없이" 류 표현 주의, "레이저" 단어 사용 주의.
- **맞춤법 필수 검사** (2024-12-11 오타 사고 재발 방지).

## 계정 운영 주의
- 잦은 로그아웃 → 2차 인증(문자 코드/백업코드) 원내 협조 필요. 백업코드 안전 보관.
- 하이라이트 커버 편집, 스토리 운영 포함.
- 외부 프로그램('애프더닥') 연동 시 Meta 외부활동 연결 필요.
