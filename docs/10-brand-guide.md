# 10. 브랜드 가이드 (로고 · 컬러 · 사용 규정)

> 2026-06-17 원내에서 브랜드 자산 5종 수령. 리브랜딩(2026-05 그랜드 리오프닝, "베러미→뷰티파크") 확정 자산.
> 모든 신규 콘텐츠는 본 가이드를 기준으로 제작한다. "베러미" 잔재 표현·구 로고 사용 금지.

## 로고 자산 (5종)

| # | 자산 | 구성 | 권장 사용처 |
|---|------|------|------------|
| 1 | **심볼마크** | 타원 프레임 + 세리프 `B` 모노그램 + 상단 4각 스파클(✦) | 프로필 아이콘(인스타/카톡/네이버), 파비콘, 워터마크, 도장형 마크 |
| 2 | **세로 락업(메인 로고)** | 심볼 + `BEAUTY PARK` + `뷰티파크의원 범어점` | 대표 로고. 배너 메인, 간판, 명함, 첫 화면 |
| 3 | **영문 워드마크** | `BEAUTY PARK` (세리프, 가로) | 가로 좁은 영역, 푸터, 콜라보 |
| 4 | **국문 워드마크** | `뷰티파크의원 범어점` | 국문 단독 표기, 의료광고 필수표기 영역 |
| 5 | **콤비 워드마크** | `BEAUTY PARK` + `뷰티파크의원 범어점` (심볼 없음) | 가로 락업이 필요한 배너 헤더/하단 띠 |

## 컬러

브랜드 핵심 컬러는 **웜 토프(greige/mocha)** 단색 + 화이트/아이보리 배경의 미니멀 조합.

| 역할 | 컬러 | HEX (⚠️ 로고 이미지 추정값 — 원본/브랜드가이드 수치로 확정 요망) |
|------|------|------|
| Primary (로고/타이틀/포인트) | 웜 토프 | **`#8C7E6E`** (대략 R140 G126 B110) |
| Primary - Deep (본문 강조) | 다크 토프 | `#6E6253` |
| Secondary (배경) | 아이보리/오프화이트 | `#F7F4EF` |
| Base | 화이트 | `#FFFFFF` |
| Text (본문) | 차콜 | `#3A352F` |

- 톤앤매너: **고급·차분·여백 많은 클린**. 채도 높은 원색·형광·과한 그라데이션 지양.
- 단색 브랜드이므로 이벤트별 포인트가 필요하면 **명도 단계(라이트 토프 ~ 다크 토프)** 안에서 변주, 외부 보색 남발 금지.

## 타이포그래피

- 영문: **세리프(Serif)** — 로고와 동일 계열의 고전적 세리프(예: Didot/Cormorant 류). 헤드라인용.
- 국문: 정돈된 **고딕/명조** 혼용. 제목은 명조 계열로 우아하게, 가독성 본문은 고딕.
- ⚠️ 정확한 폰트명은 원본 파일에서 확인 필요(추정). → 자체 제작 시 위 분위기를 맞춘 대체 폰트 사용.

## 사용 규정 (Do / Don't)

**Do**
- 충분한 여백(클리어 스페이스) 확보 — 심볼 높이의 최소 0.5배.
- 단색 배경(화이트/아이보리/토프) 위에서 사용.
- 작은 크기에서는 **심볼마크 단독** 또는 **콤비 워드마크** 사용(세로 락업은 가독성 저하).

**Don't**
- 로고 색 임의 변경, 그림자/외곽선/입체효과 추가 금지.
- 비율 왜곡(가로/세로 늘이기) 금지.
- 복잡한 사진 위에 직접 얹지 말 것(반투명 패널/딤 처리 후 사용).
- "베러미" 구 로고·구 색상과 혼용 금지.
