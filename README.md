# beautypark-librarylab

뷰티파크의원 범어점(구 베러미의원 범어점)의 마케팅 운영을 위한 **운영 지식 베이스(플레이북)**입니다.

## 배경

- 2024년 8월부터 마케팅 대행사 **라이브러리랩(라랩)**이 인스타그램·카카오톡 채널·메타광고·월간 이벤트 기획/디자인·웹 배너·리포트를 대행해 왔습니다.
- 2026년 8월 라이브러리랩과의 **계약 종료** 예정이며, 이후 동일한 업무를 **내부(에이전트 운영)로 전환**하는 것이 목표입니다.
- 이 레포는 라이브러리랩이 수행하던 업무를 **그대로 재현**할 수 있도록 프로세스·규격·계정·콘텐츠 자산을 문서화합니다.

> 핵심 미션: **이벤트 생성 → 웹사이트 배너 게시 → 인스타그램 홍보**를 매월 안정적으로 운영하는 것.
