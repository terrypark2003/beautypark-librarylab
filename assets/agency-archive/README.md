# 업체(라이브러리랩) 제작물 아카이브 = 디자인 학습 코퍼스

목적: 업체가 만들어준 **모든 완성 산출물**을 모아, 이벤트별 디자인 패턴을 그대로
재현하는 **템플릿 라이브러리**로 변환한다. (입력 엑셀 → 업체 스타일의 포스터/영상 자동 생성)

> "학습"의 실제 의미: 별도의 머신러닝 모델 훈련이 아니라, 에이전트가 이 산출물들을 분석해
> **반복되는 레이아웃·테마·카피·가격 표기 규칙**을 추출하고 HTML 렌더링 템플릿으로 코드화하는 것.
> (한글·가격을 100% 정확히 찍어야 하는 의료광고 특성상, AI 이미지 생성이 아니라 템플릿 렌더링이 맞다.)

## 여기에 넣어주세요
```
posters/   완성 이벤트 이미지 (월별·이벤트별 PNG/JPG)  ← 가장 중요
excels/    월별 '디자인 작업 요청서' 엑셀 (마스터 통합본 포함)
videos/    디스플레이/홍보 영상 (.mp4 / .3gp)
```
- 파일명은 가능하면 `YYYY-MM_이벤트명.png` 형태면 분류가 쉽습니다(아니어도 됨).
- 인스타 카드뉴스, 홈페이지 배너/팝업 등 형태가 달라도 전부 넣어주세요(형태별 규격 학습).

## 에이전트가 추출하는 것 (→ 템플릿화)
1. **반복 테마** — 월초 매거진(시즌), 화수목(평일 한정), READY TO GO(하늘), 카톡 친구전용(버블), 상반기 결산(보드) 등
2. **레이아웃 규칙** — 로고 위치, 타이틀 구성, 흰 카드 행, 정상가 취소선 + 이벤트가 만원, 푸터
3. **가격 표기** — 이벤트가(부가세 전) 만원 강조 / 정상가 취소선 / "부가세 10% 별도, 현금·카드 동일"
4. **카피 톤·해시태그** — 월 테마 문구, 강조(볼드) 상품 규칙

## 현재 진행 상황
- ✅ 렌더 PoC: `tools/render-poster.mjs` (엑셀 파싱 → 테마 적용 → PNG). 샘플: `outputs/2026-06/`
- ⬜ 업체 완성 포스터 업로드 대기 → 테마별 템플릿 정밀화
- ⬜ 영상 템플릿(디스플레이용) — 이미지 안정화 후 착수
