규정·컬러·사용법은 [`../../docs/10-brand-guide.md`](../../docs/10-brand-guide.md) 참조.

## 메모
- 가능하면 **SVG(벡터)** 원본을 받아 보관하세요. 확대/리사이즈·색상관리에 유리하고, 자동 제작 파이프라인에서 재사용하기 좋습니다.
- PNG만 있을 경우 **투명 배경 + 충분한 해상도(긴 변 2000px+)** 로 보관.
- 정확한 브랜드 컬러 HEX와 폰트명을 함께 받으면 `docs/10-brand-guide.md`의 추정값을 확정값으로 교체하세요.
- 채팅으로 업로드한 이미지는 자동으로 이 폴더에 저장되지 않습니다 — 원본 파일을 직접 커밋해 주세요.
